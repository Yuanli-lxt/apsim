# 齐河县连续模拟预热与初始矿质氮敏感性试验

## 目的

检验连续模拟经过多年土壤水氮循环后，产量与播前矿质氮是否不再明显依赖缺乏实测依据的初始 NO3/NH4。该试验不用于优化品种或作物生理参数，也不把施肥情景拟合为年度统计产量。

## 因子设计

- 连续时段：2010-10-01 至 2023-12-30；2010—2014主要形成预热状态，2015—2023为正式比较年份。
- 空间分辨率：10 km，共88个独立土壤—气象案例、25个AgERA5节点、14个HWSD土壤单元。
- 初始矿质氮：模板 NO3 和 NH4 浓度同时乘以0.5、1.0、1.5；这是初值敏感性边界，不代表三个观测等级。
- 施氮方案：固定180、统计约束中央值、中央值的0.75倍、中央值的1.25倍。后两者是探索性边界，所以实际为4个管理成员、12个组合、1056个连续APSIM案例。
- 小麦季氮肥按播种/返青拔节60%/40%分施，玉米按播种/追肥1/3和2/3分施，肥料类型为 `urea_N`。灌溉、播种窗口、土壤、品种及作物参数保持普通农户基线不变。

固定180 kg N/ha仅用于复现旧测试条件，不是必须维持的管理参数。统计约束中央值按“折纯氮 + 复合肥折纯量×1/3”估算县域氮总量，再除以全部农作物播种面积。该值约束年度数量级，但复合肥氮比例、作物间分配和施用日期均不是官方表直接观测。

## 数据真实性与保留差异

- 每个年份和气象节点使用对应的AgERA5逐日辐射、降水、最高温和最低温，不循环复制某一年的天气。
- 每个案例保留原HWSD土壤剖面、容重、水力性质和土壤单元差异；只有初始NO3/NH4按预设倍数改变。
- 土壤水、矿质氮、有机质和残茬在年际之间连续传递，不逐年重置。
- 2012—2023的县级化肥总量来自德州统计年鉴。2010—2011因当前官方在线连续资料从统计年度2012起，沿用2012中央强度作为预热填补，必须标为假设。
- 逐年作物掩膜和逐年县级播期、施肥日期、灌溉量尚未获得；当前仍使用2020轮作空间范围与统一普通农户管理。它们是模型假设，不能表述成真实年度管理。

2012—2014证据表见 `data/processed/spatial/county_pilot_2020/calibration/qihe_warmup_fertilizer_history_2012_2014.csv`，2015—2023统计表见 `qihe_multiyear_official_fertilizer_statistics.csv`。

## 预先声明的收敛判据

同一施氮方案内，对三个初始氮倍数比较：

- 年度作物产量最大相对极差不超过2%；
- 10月1日剖面 NO3+NH4 最大相对极差不超过10%。

两个条件同时满足才称该年度对初始氮达到操作性收敛。阈值是在查看结果前设定的；即使产量先收敛而矿质氮未收敛，也不能说初值影响已经完全消除。

## 运行顺序

```powershell
python scripts/spatial_data/download_agera5_qihe.py --years 2010 2011 2012 2013 2014 2015 2016 2021 2022 2023 --workers 4
python scripts/spatial_data/prepare_qihe_warmup_weather.py --run-id qihe_warmup_2010_2023_v1
python scripts/apsim_inputs/run_qihe_warmup_sensitivity.py --run-id <new_run_id> --data-run-id qihe_warmup_2010_2023_v1 --workers 4
python scripts/apsim_inputs/analyze_qihe_warmup_convergence.py --run-id <new_run_id> --data-run-id qihe_warmup_2010_2023_v1
```

每次使用新 `run_id`。原始 `.apsim`、APSIM输出、日志、案例状态、年度案例结果及元数据均保留在 `outputs/spatial/county_pilot_2020/warmup_sensitivity/<run_id>/`。

## 2026-07-16完整10 km运行结果

正式运行号为 `qihe_warmup_10km_full_2010_2023_20260716_v1`，气象输入运行号为 `qihe_warmup_2010_2023_v2`。APSIM Classic版本为7.10.4221.0。

- 1056/1056个连续案例成功，耗时2443.44秒；共13728条案例—年度记录，关键产量和NO3/NH4字段无缺失。
- 每个管理—初始氮组合均有88案例；县域代表轮作面积58,918.84 ha，覆盖28个10 km格网、14个HWSD土壤单元和25个AgERA5节点。
- 2015—2023四个管理成员全部达到预设双重收敛标准。跨三个初始氮水平的最大年度相对极差为：小麦0.53%、玉米0.23%、10月1日矿质氮2.90%，均低于2%和10%的阈值。
- 2011年四个管理成员均未完全收敛；2012年固定180情景的矿质氮仍未通过；2014年统计中央情景的小麦相对极差为3.60%，出现一次非单调未通过。因而不能宣称经过固定两三年后所有情景永久收敛，只能说从预先规定的2015正式期开始，本试验内的初始氮影响持续低于操作阈值。
- 在1.0倍初始氮下，2015—2023统计中央情景的县域平均小麦/玉米产量为6038.5/12242.7 kg/ha；低边界为5270.9/10955.5 kg/ha，高边界为6378.3/13082.6 kg/ha。管理情景差异明显大于预热后初始氮差异。
- 统计中央情景的10月1日矿质氮从2015年的376.7 kg/ha降至2022年的29.4 kg/ha，2023年为42.7 kg/ha。这反映年度统计约束施氮下降与模型连续氮循环的共同结果，提示后期可能出现氮供给趋紧；它不是土壤实测验证。

完整运行目录约3.21 GiB。`run_file_manifest.csv`列出6344个生成文件的大小、修改时间和SHA-256；`run_metadata.json`记录配置、APSIM可执行文件、模板和模拟单元清单的哈希。原始逐日APSIM输出保留，县域汇总见 `annual_county_yield_and_mineral_n.csv`，逐年收敛见 `warmup_convergence_by_year.csv`。
