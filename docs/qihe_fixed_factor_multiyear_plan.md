# 齐河县固定乘法系数多年验证实施计划

## 研究边界

本阶段冻结 APSIM Classic 模型结构、土壤剖面、普通农户管理、既有小麦和玉米品种参数以及 2020 年轮作掩膜。研究对象是县域统计后校正系数的跨年稳定性、尺度可迁移性和不确定性，不使用品种或作物生理参数拟合统计产量。

2020 年 5 km 固定系数定义为“2020 年正式县级单产 / 2020 年 5 km 原始 APSIM 面积加权单产”：小麦 1.0038739284，玉米 0.6040688345。2020 年校正残差为零是数学定义，不是验证结果。该系数只改变绝对量级，不改变格网排序、空间变异系数或年际响应的方向。

## 数据可用性审计（2026-07-16更新）

- 已核验正式县级统计：2015—2023 年连续序列。数据均来自德州市统计局官方统计年鉴栏目及其PDF附件；逐年记录统计年份、出版年份、表名、PDF页码、印刷页码和下载地址。
- 2015、2016、2021年的年鉴PDF分别存在文本编码损坏或扫描图像问题，采用原始表页视觉复核；2017—2020、2022—2023同时使用可提取文本和表页复核。`qihe_multiyear_statistics_gaps.csv` 当前仅保留表头，表示目标区间内无统计缺年。
- 已有 AgERA5：2017—2020 年。它支持带预热的2018—2020完整轮作季；不能支持2017冬小麦，因为缺少2016年秋播及预热气象。
- 已有原始 APSIM：2018—2020 年，5 km 和10 km；2020轮作掩膜和统一普通农户管理在三年间固定。

正式统计现已满足2015—2023扩展目标，但当前完整“正式统计 + 原始模拟”交集仍为2018—2020。下一阶段应补齐至少2014—2023 AgERA5，以便为2015冬小麦保留2014年秋播和前期状态形成期；在新增原始APSIM完成前，三年留一年交叉验证和 Bootstrap 仍只能作为探索性诊断。

## 顺序与验收条件

1. **冻结基线。** 记录 Git 状态、APSIM可执行文件、模型模板、管理配置、气象、土壤、掩膜、脚本和既有结果的路径、大小、时间与 SHA-256；复制县域和格网原始汇总到新 run 目录。验收：5/10 km 均为2018—2020，案例成功数分别为101/101和88/88。
2. **扩展统计。** 只录入已核验的正式县级年鉴值，区分统计年和出版年，量化面积×单产与总产量的舍入差。验收：作物×年份唯一、单位换算一致、缺年单列。
3. **扩展气象和模拟。** 在取得新增年份正式统计与完整预热气象后，扩展现有 AgERA5 下载、MET生成和 APSIM运行脚本；先10 km诊断，关键年5 km复核。当前不因统计缺口盲目批量下载和运行。
4. **固定系数诊断。** 对原始和2020固定系数结果逐年计算偏差及多年指标；计算隐含系数、趋势、连续偏差和年际变化诊断。独立验证范围严格标为2018—2019。
5. **年份交叉验证。** 以零截距最小二乘在训练年估计系数；轮流留年，并与原始、2020固定系数和全样本描述性系数比较。全样本拟合结果不得称为独立验证。
6. **不确定性。** 年份重采样 Bootstrap 比较隐含系数均值、零截距最小二乘和零截距稳健回归；报告95%区间和逐年产量区间，并做删一年影响分析。
7. **掩膜与管理。** 固定系数研究完成且获得逐年资料后再实施；当前统一管理明确为模型假设。
8. **空间输出。** 10 km 用于批量诊断，5 km用于最终复核和制图。缺少乡镇/地块/样方观测时只称“模拟空间异质性”，不称“空间验证”。

## 停止使用固定系数的判据

若新增独立年份显示连续同号偏差、误差随年份显著漂移、隐含系数变异系数或区间过宽、固定校正不改善年际变化、滚动预测显著劣于原始模型，或尺度/面积口径改变导致不可解释偏差，应停止把单一固定系数用于预测，转向按可观测协变量定义的动态校正、分层误差模型或重新检查气象—土壤—管理机制。该判断不能通过逐年单独拟合系数来规避。
